MUDA - Multiple Upload Directory Assistant (alternatively: 無駄)

Muda is a directory dumper script originally written for 8chan. Given a directory, a thread, and a board, it will dutifully post all images in the directory automatically, leaving you to do other things.

Requirements

  • Ruby 2.1.x
  • Bundler

Instructions

  • Install bundler (gem install bundler) to manage dependencies
  • Kick off bundler to install dependencies: bundle install
  • In case of errors, you're probably having trouble compiling the stuff that comes with Mechanize.
  • Run the script with the help option to see full details: ./muda.rb --help
  • Please report any bugs here :)

Usage

Minimum required: ./muda.rb --thread 123456 --board b --directory /home/anon/images

Other options are available and mostly self-explanatory. Examine the output of --help to customize your dump by setting name, email, body text, and more.

Compatibility

Should work with any vichan/infinite based site by default. Other sites will require mucking about with the code that locates and fills the post form.
